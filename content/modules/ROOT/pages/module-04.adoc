The following command will add those lines for you.

```bash
tee -a /root/hosts << EOF
      - port: ['9999/tcp']
        state: enabled
EOF
....

Now run the playbook in the `+rhel+` terminal.

[source,bash]
----
ansible-playbook -i hosts -b firewall.yml
----
